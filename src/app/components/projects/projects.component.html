Smart Pond Monitoring System

Manual pond monitoring is inefficient and lacks real-time insights, leading to poor water quality management. The Smart Pond Monitoring System addresses this by automating sensor-based tracking of water conditions through a web application for efficient and timely decision-making.

  • ✅ Developed a Full Stack Web Application using Angular, Spring Boot, and MongoDB.
  • ✅ Implemented real-time monitoring of water level, temperature, humidity, and pH sensors.
  • ✅ Designed an admin module for adding ponds, managing sensors, and modifying parameters.
